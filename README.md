# GigFlow 

GigFlow is a mini freelance marketplace where users can post gigs and freelancers can bid on them.

## Tech Stack
- Frontend: React (Vite) + Tailwind CSS
- Backend: Node.js + Express.js
- Database: MongoDB
- Authentication: JWT (HttpOnly Cookies)

## Features
- User authentication
- Gig posting & browsing
- Bidding system
- Hiring logic with atomic updates
